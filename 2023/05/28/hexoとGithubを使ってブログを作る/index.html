Github

リポジトリの作成

注意点

  • publicで作成する必要がある。
    privateで作成した場合、Github pagesで世界中に公開することが出来ない。
    特定の人だけが見れる形で公開する場合(社内サイトなど)はprivateで作成する。(要課金)
  • Repository nameをxxx.github.ioで作成する必要がある。
    1. ‘Owner.github.io’（おすすめ）
      Githubのアカウント名を先頭に付けてrepositoryを作成した場合、公開後のurlがOwner.github.ioとなる。
    2. ‘xxx.github.io’
      Githubのアカウント名以外を先頭に付けてrepositoryを作成した場合、公開後のurlがOwner.github.io/xxx.github.ioとなる。

sshキーの作成

注意点

  • id_rsa(default)の名前以外で作成した場合はconfigファイルを作成する必要がある。
    configファイルの中身に
    Host github github.com
    HostName github.com
    IdentityFile ~/.ssh/”ファイル名”
    User git
    を追加する。

sshキーの登録

注意点

  • 一つのsshキーを複数のGithubアカウントに登録できない。
    sshキーが複数必要なので、sshキーのconfigファイルを作る必要がある。
